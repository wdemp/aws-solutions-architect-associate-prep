# S3
-Simple Storage Service
-Object Storage-max object size is 5 terabytes
- S3 features
  - Cross Region Replication 
  - Lifecycle Policy
  -Encryption 
  -Static Website Hosting
  
Bucket: a container (web folder) for objects (files) stored in Amazon S3.  They are are global names; menaing there can't be any duplicates with the same name. The containers can contain up to 63 lowercase letters.
